# A* Algorithm in 3 dimensions

## Installation

Before you can run the build process you need to install typescript.
You can do this with `npm i -g typescript` or `yarn global add typescript`.

Yarn installation:
```
yarn install
yarn build (optional)
yarn start
```

NPM alternative:
```
npm install
npm run build (optional)
npm run start
```
